# Bundled demo ROM

## Tobu Tobu Girl Deluxe (`tobudx.gb`)

An open-source arcade platformer for Game Boy / Game Boy Color / Super Game Boy,
auto-loaded as the default demo so the emulator shows something the moment the
page opens. Renders in full CGB color.

- **Author:** Simon Larsen (DX remaster); original *Tobu Tobu Girl* by Tangram Games
